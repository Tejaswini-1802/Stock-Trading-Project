# 📊 Stock Analysis Project

## 🚀 Features
- Data from yfinance (RELIANCE.NS)
- Technical Indicators (RSI, MACD)
- Trading Strategy
- Machine Learning Model
- Streamlit Dashboard

## 🛠 Tech Stack
Python, Pandas, Scikit-learn, Streamlit

## 📈 Output
- Buy/Sell Signals
- Model Predictions

## ▶️ Run
pip install -r requirements.txt  
python src/main.py  
streamlit run app/app.py
